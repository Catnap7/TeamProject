가상주소 http://localhost:8080/myweb/
=> 웹서버 => WAS
=> request,response 객체 생성  request요청정보 저장
=> web.xml 참조
=> 가상주소 /    
=> appServlet  => 찾아
=> org.springframework.web.servlet.DispatcherServlet
=> /WEB-INF/spring/root-context.xml 실행
=> /WEB-INF/spring/appServlet/servlet-context.xml 실행

=> <annotation-driven /> @자동인식
=> com.itwillbs.myweb 자동스캔
=> @Controller 인식
=> @RequestMapping(value = "/", method = RequestMethod.GET) 인식
=> home()메서드 동작 
=> model 데이터 저장 =>  home.jsp 이동

가상주소 http://localhost:8080/myweb/home2
가상주소 "/home2"  => home2() 실행 => home2.jsp 이동
